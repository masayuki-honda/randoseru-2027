## ✅ 確定事項・購入の注意点

- **早期予約推奨**：人気メーカー・人気色は2〜3月に完売することがある（特に池田屋）
- **試着は必須**：背カンの良し悪しは実際に背負って歩いてみないとわからない
- **保証の確認**：「6年間無償修理」の適用範囲はメーカーにより差がある
- **本革リメイク**：「卒業後に財布になる」はセールストーク。実施家庭は少数派、費用1.5〜3万円

## 📊 参考：市場データ（Wikipedia ランドセル工業会調査）

- 2024年入学の平均購入価格：**約59,000円**（2018年比 +約7,800円）
- 素材シェア：クラリーノ等 人工皮革 **約70%** ／ 本革 約30%
- **ランドセル症候群**：通学時の荷物が3〜5kgになり身体的負担が出る問題。小学生の約7割が経験。背カン設計と「置き勉」許可が主な対策。
